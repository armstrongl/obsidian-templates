# Web Clipper Templates

## Properties

Default properties:

- `title`: `{{title|title}}`
- `author`: `{{author|title}}`
- `published`: `{{published|date:"YYYY-MM-DD"}}`
- `created`: `{{date|date:"YYYY-MM-DD"}}`
- `description`: `{{description}}`
- `tags`: `clipping, {{site|kebab}}, {{tags|kebab}}`
- `url`: `{{url}}`
- `image`: `{{image}}`
- `source`: `{{url}}`
